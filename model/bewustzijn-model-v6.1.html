MODEL NOTES v6.1
Quantum Experiential Dynamics — an open quantum systems architecture for experiential state evolution

State update: Psi = (A_w + gamma*A_retention + delta*A_h + beta_g*A_group + beta_e*A_extern) / |...|, combined in the half-angle (Bloch) Hilbert space, consistent with the density matrix

Key mechanisms (thresholds emergent; scales tied to character):
(1) Qubit density matrices on Russell's circumplex, Bloch-equator convention: opposite emotions orthogonal, ambivalence = von Neumann entropy.
(2) Route choice from one measure: trace distance T between perception and memory (Helstrom). Recognition if T < own self-fluctuation; surprise if T > own p90; deliberation (Route B) between.
(3) Emotional intensity = interference visibility (half-angle): aligned sources reinforce, orthogonal (opposite) sources damp; ambivalence goes to the perpendicular (tense), not to null.
(4) gamma = P*F (retention, Husserl); delta = Bloch length of memory rho; character rho0 decays with dynamic lambda.
(5) Group affinity = trace distance between dispositional rhos, threshold = 1.5x mean self-fluctuation; beta_g ≤ P/sqrt(group), beta_e ≤ P/sqrt(N).
(6) Route B = quantum trajectory on a qutrit (perception / conflict / limbo): H drives ambivalence (endogenous conflict), the environment measures at rate r; decision time EMERGENT (no iteration cap); indecision = limbo, a located non-empty outcome. Leggett-Garg K3 > 1 (no classical model reproduces it) — a testable prediction. K (ambivalence sensitivity) emergent from character.
(7) Groundedness layer (grounded <-> ungrounded), per entity, monostable (recovers to trait baseline); mood colours the FELT valence (readout, does not feed the dynamics). Classical layer on the quantum core.
