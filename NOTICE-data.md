# Third-party data sources (referenced, not redistributed)

1. Confidence Database: Rahnev, D. et al. (2020). The Confidence Database.
   Nature Human Behaviour, 4, 317-325. Data: https://osf.io/s46pr/
   Datasets used in our analyses: Adler 2018 (E1-E3), Arbuzova 2021, Bang 2019,
   Hellmann 2023, Konishi 2019.
2. EmotionTimeSeries archive: Haslbeck, J.M.B. https://github.com/jmbh/EmotionTimeSeries
   (OSF: https://osf.io/4czv3/). Datasets used: Bringmann 2013, Bringmann 2016, Rowland 2020,
   Wright 2017 (as shared via Wendt et al. 2020), Fisher 2017, Fried 2021.
   Vrijen 2018 was excluded because reuse requires the original authors' permission.
3. Mental rotation RT data: Lindelov, J.K. https://github.com/lindeloev/shiny-rt (mrt_data.csv).

All analyses on these data are secondary analyses of openly shared, anonymised datasets.
Obtain the data from the original sources; our scripts expect the original file layouts.
