Third-party data sources (referenced, not redistributed)

None of the datasets below are included in this repository or in the Zenodo archive. All analyses are secondary analyses of openly shared, anonymised datasets; only derived aggregate statistics (per-participant summary values, histograms) appear in the paper figures. Obtain the data from the original sources; our scripts expect the original file layouts. License status verified 21 July 2026.

1. Confidence Database

Rahnev, D. et al. (2020). The Confidence Database. Nature Human Behaviour, 4, 317-325. Data: https://osf.io/s46pr/ — License: CC0 1.0 Universal (public domain). Datasets used in our analyses: Adler 2018 (E1-E3), Arbuzova 2021, Bang 2019, Hellmann 2023, Konishi 2019 (motion and colour). Figure 1 (human panels) is derived from Adler 2018 under this license.

2. EmotionTimeSeries archive

Haslbeck, J.M.B. https://github.com/jmbh/EmotionTimeSeries (OSF: https://osf.io/4czv3/). The archive repository carries no explicit license file. It is a compilation of datasets that were openly shared by their original authors (acknowledged as such in the archive's README). We therefore rely on the openness of the original sources, use the data for non-commercial secondary analysis only, do not redistribute any records, and show only derived per-participant statistics (inertia and variability values, Figure 4). Datasets used: Bringmann 2013 (data shared with a PLoS ONE publication, CC BY), Bringmann 2016, Rowland & Wenzel 2020, Wright 2017 (as shared via Wendt et al. 2020), Fisher 2017, Fried 2021. Vrijen 2018 was excluded because the authors ask that reuse be requested; we did not request it and did not use the data.

3. Mental rotation RT data

Lindeløv, J.K. https://github.com/lindeloev/shiny-rt (mrt_data.csv). The repository carries no explicit license file. These data were used only for early exploratory distribution checks and do not appear in the paper or its figures; the published decision-time comparison (Figure 1) uses exclusively the CC0 Confidence Database.

Summary of the compliance position
Nothing redistributed; links and citations only.
Published figures rest on CC0 data (Figure 1) and on derived aggregate statistics of openly shared data (Figure 4), which are facts, not copies of the protected records.
The one dataset with an explicit permission requirement (Vrijen 2018) was excluded.
